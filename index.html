Surveillance LLM

Surveillance LLM integrates the image description capabilities of BLIP (Bootstrapping Language-Image Pre-training for Unified Vision-Language Understanding and Generation) and the text understanding and reasoning capabilities of ChatGPT. When an image is captured by a camera, feed into the system. The system processes it and uses BLIP to describe the contents of the image. It then leverages ChatGPT to analyze the image description and detect any potential security threats and make announcement using google text to speech library.
